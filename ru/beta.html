KyBook v0.9.0 готов для открытого бета теста.

Сборка 900
  • Добавлен показ отзывов для некоторых OPDS каталогов.
  • Исправлена проблема с показом отчета об ошибке после неудачной попытки отправить сообщение в vk.com.
Сборка 901
  • Улучшена функция выделения текста.
  • Исправлена проблема с выделением текста в некоторых PDF.
  • Исправлено редкое аварийное завершение при выделении текста в PDF.
  • Исправлен редкое аварийное завершение при повороте экрана в процессе чтения.
  • Добавлена поддержка для функции быстрой выборки в PDF, DjVu.
  • Улучшена функция быстрой выборки текста.
  • Улучшена поддержка поиска в некоторых OPDS каталогах.
  • Улучшена поддержка метаданных DjVu.
Сборка 902
  • Исправлена проблема с невозможность поставить закладку на PDF страницу не содержащую текста.
  • Добавлена возможность пометить книгу как прочитанную.
  • Добавлена сортировка по прогрессу чтения в разделе Книги.
  • Добавлена возможность добавить ваше мнение о книге через карточку книги.
  • Добавлена функция для подсчета количества слов в EPUB, MOBI, FB2 книгах.
  • Добавлена возможность изменить цвет закладки.
