--- layout: kybook_news title: Beta v0.9.2 (build 922) ---

Version 0.9.2 is available for public beta-testing.

What's new

Build 921
  • Added support for shared folders on the local network (SMB/CIFS).
Build 922
  • Fixed an issue with hanging opening EPUB, FB2 books in some cases.
  • Fixed several bugs causing crashes.
  • Removed an ability to select the Inbox folder (readonly) as the destination when copying or moving files.
  • Improved the synchronization of the books metadata via iCloud.
  • Added synchronization of bookmarks and notes by the forced synchronization in the open book.
  • Added an option for disabling updates in the open book when receiving push notifications about changing of a read position, adding or deleting of bookmarks and notes.
  • Added an option for synchronization of the user's content in the Files/App folder via iCloud.
