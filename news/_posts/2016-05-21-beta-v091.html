Version 0.9.1 is available for public beta-testing.

What's new

Build 910
  • Redesigned search in the Files section.
  • Added an ability to update a folder in the Files via the pull-to-refresh gesture.
  • Added an ability to reset the subfolder settings into the parent's folder settings.
  • Added a sidebar with the siblings folders in the Files section in a landscape orientation.
  • Added a feature to grouping by author/series in the Files section.
  • Fixed a bug with recursive coping of a folder into itself.
  • Fixed an issue when unable to open certain fb2 and cbr books from the Cloud Storage.
  • Improved support for some OPDS catalogs.
Build 911
  • Added an ability to edit the reading themes for PDF,DjVu.
  • Added an ability to specify a gradient background in a theme for EPUB,MOBI,FB2 (the Sand theme includes gradient background by default).
  • Added support for TAR archives.
  • Added support for comics in cbt format.
  • Added selection of permissions to access the files when added Dropbox (the connections with different access modes is allowed simultaneously).
