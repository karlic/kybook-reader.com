Version 0.9.5 is available for public beta-testing.

What's new

Build 950
  • Partial German localization.
Build 951
  • Added experimental support for Cloud Mail.ru .
Build 952
  • Improved support for MOBI format.
  • Improved search of OPDS catalogs on the local network.
  • Added validation of OPDS catalog when added manually.
  • Added support for Google Books as a library.
  • Added support for the Samizdat site as library.
  • Added search for covers through Open Library.
  • Fixed some bugs causing crashes the application.
  • Fixed various minor bugs.
