--- layout: kybook_news title: Beta v0.9.4 (build 943) ---

Version 0.9.4 is available for public beta-testing.

What's new

Build 940
  • Fixed an issue with a guest login to a shared network folder.
  • Fixed an issue with the inability to use a shared network folder without specifying a connection point.
  • Fixed an issue with an broken translation and the dictionary lookup for some EPUB, MOBI ebooks.
Build 941
  • Added the ability to tunnel connection to OPDS catalog via the Tor network.
Build 943
  • Fixed an issue when unable to start play the next chapter of an audiobook if the application in the background.
  • Added correction of the broken encoding (win cp1251) in id3 tags of mp3 files.
  • Added downloadable fonts.
  • Changed the application icon.
  • Fixed various bugs.
