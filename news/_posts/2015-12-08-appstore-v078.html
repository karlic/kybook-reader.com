Version 0.7.8 available on the App Store.

What's new

  • Added support for MobiPocket (mobi) and Kindle (azw3) formats.
  • Added the toolbar for notes in a book.
  • Added the navigation bar for a search results.
  • Completed the Fuzzy search and search using regular expressions in a book.
  • Added support for password-protected PDF files.
  • Added support for password-protected comics in cbr format.
  • Added the ability to specify the text encoding for txt files.
  • Added the ability to specify MobiPocket PID / Kindle serial number for mobi/azw3 files.
  • Fixed an issue with the editing of metadata in a book card when a book is open.
  • Fixed an issue with the showing of found search phrase in a book.
