--- layout: kybook_news title: Beta v0.9.3 (build 934) ---

Version 0.9.3 is available for public beta-testing.

What's new

Build 933
  • Added support for dictionaries in StarDict format.
  • Reworked and optimized support for dictionaries in Lingvo DSL format, now it supports very large dictionaries.
  • Added a fuzzy search in the dictionaries.
  • Added an ability to change the order of searching in dictionaries.
  • Added downloadable free dictionaries.
  • Changed the application icon.
Build 934
  • Added an option for enabling the support for swipe gesture on the screen edge for accessing of the side menu.
  • Redesigned the color picker dialog in the notes and themes.
