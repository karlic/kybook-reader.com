Version 0.8.1 (812) is available for public beta-testing.

What's new

  • Added new themes: Sand, Tango.
  • Added the ability to simultaneously edit the metadata of several books.
  • Fixed an issue when unable to remove a book's sequence from a metadata.
  • Improved editing of the book sequence in a book card.
  • Fixed a bug with the inability to connect Samba server without specifying the sharepoint.
  • Added auto-scroll feature for epub/fb2/mobi formats.
