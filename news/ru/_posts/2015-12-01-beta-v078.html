Версия 0.7.8 готова для открытого бета теста.

Что нового

  • Добавлена панель инструментов для заметок в книге.
  • Добавлена панель навигации по результатам поиска.
  • Доделан нечеткий поиск и поиск с помощью регулярных выражений в книге.
  • Добавлена поддержка для PDF документов защищенных паролем.
  • Добавлена поддержка для комиксов в формате cbr защищенных паролем.
  • Добавлена возможность указать кодировку текста для книг в формате txt.
  • Добавлен возможность указать MobiPocket PID / Kindle serial number для книг в форматах mobi/azw3.
  • Исправлена проблема с редактированием метаданных в книжной карточке в открытой книге.
  • Исправлена проблема с показом найденной фразы при поиске в книге.
