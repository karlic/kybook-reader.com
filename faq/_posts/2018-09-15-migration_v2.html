How to migrate to KyBook 3 from KyBook 2

Requirements

Firstly make sure that you have the most recent versions of KyBook 2 and 3. If your versions are outdated, please update them from the App Store.

The minimum versions of KyBook are:

  • KyBook 3 must be at least v0.5.2
  • KyBook 2 must be at least v1.2.4

To check your KyBook versions, please open the Control/About screen in KyBook 3, by tapping the gear symbol on the home screen and scrolling down to About and the Support screen in KyBook 2.

Migrating

To start migration, launch KyBook 2, go to the Support screen and tap on Copy books to KyBook 3. KyBook 2 will calculate the required disk space and ask you to confirm migration. On confirmation, KyBook 2 will copy the database and books from KyBook 2 into KyBook 3 and launch KyBook 3 for processing the copied books. Processing books takes some time, so please be patient.

Not all information can be transferred without loss, some notes may be a little displaced and not all the e-books' metadata from KyBook 2 may be supported in KyBook 3. KyBook 2 and KyBook 3 are different apps with different technology stacks and handle e-books in different ways, so some compatibility problems are unavoidable.

Important remarks about Cloud Storage (Dropbox, Google Drive, iCloud Drive, Yandex Disk, etc.)

During migration, KyBook 3 downloads books located in Cloud Storage for processing, which takes significant disk space and time. The best option is to link your Cloud Storage manually in KyBook 3, before migration. However, this is optional as KyBook 3 can automatically link Cloud Storage using credentials from KyBook 2.

KyBook 3 can't migrate books from iCloud Drive because KyBook 3 now uses the KyBook 3 iCloud folder and has no access to the former iCloud KyBook folder.

For the same reason, KyBook 3 can't migrate books from the Yandex Disk App Cloud Storage (the common Yandex Disk Cloud Storage has no problem with migration).

To workaround the issue with migration from iCloud Drive and Yandex Disk App, first manually copy your books from the Cloud Storage into any folder on the device. To do that, open the Files screen, then open a Cloud Storage, select folders containing your books (by long tapping on a folder and tapping the menu Select item ) and then tap the Copy button on the bottom bar. After copying books from Cloud Storage, run migration as described above.
