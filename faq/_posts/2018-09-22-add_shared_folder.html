How do I access a shared folder on my home server from KyBook 3?

With KyBook 2 I was able to access a share folder on my home server through WiFi to add books. I don't see that option in KyBook 3.

The option is there. On the home screen:

  1. Tap the big red Add button (see image below)
  2. Tap Folder
  3. Tap Add SMB/CIFS Share
