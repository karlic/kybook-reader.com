When opening a book from cloud storage such as Google Drive, does KyBook 3 stream or download the book to local storage?

I should like to stream books directly from the cloud without downloading the whole library onto my phone.

KyBook 3 currently only streams audio books from cloud storage.

It is possible that PDF and DjVu book formats will be streamable in a future release.

User can toggle streaming for audio books on the KyBook Control/Network screen.
