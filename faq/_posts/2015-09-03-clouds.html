--- layout: kybook_post title: Cloud Storage, WebDAV ---

Version 1.0.6 supports the following Cloud Storage:

  • iCloud
  • Dropbox
  • Яндекс Диск
  • Google Диск
  • OneDrive
  • Cloud Mail.ru (limitedly)

Moreover, you can add any cloud storage supports the WebDAV protocol, such as:

  • Box
  • 4shared
  • myDrive
  • ownCloud based
