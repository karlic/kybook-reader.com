--- layout: kybook_post-ru title: Миграция с KyBook 1 ---

Для удобства перехода на KyBook 2, пользователь может включить импорт метаданных с первой версии приложения.

Для переноса книг с метаданными проще всего скопировать папки с книгами из KyBook 1 через iTunes Общие файлы на Ваш ПК и затем добавить ранее скопированные папки в KyBook 2 с помощью iTunes Общие файлы. После переноса папок с книгами в KyBook 2, достаточно подключить соответствующие добавленные папки на экрана 'Мои файлы'.

Другой способ подходит только для пользователей Про версии KyBook 1. Предварительно необходимо синхронизировать KyBook с одним из Ваших облачных хранилищ (Dropbox или Яндекс Диск) запустив про-функцию синхронизации, и по завершении синхронизации, необходимо подключить в KyBook 2 соответствующее облачное хранилище, зайти в папки с книгами в облачном хранилище позволив KyBook 2 импортировать метаданные версии 1.

При наличии в папке с книгами подпапки c метаданными из первой версии KyBook по имени #metadata#, метаданные (название, авторы, серия и т.п.) будет взяты из метаданных первой версии а не получены из файла книги. В настоящий момент импорт ограничен, не все значения метаданных первой версии могут быть импортированы.
