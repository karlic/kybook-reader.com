--- layout: kybook_post-ru title: Миграция с KyBook 1 ---

Для удобства перехода на KyBook 2, пользователь может включить импорт метаданных с первой версии приложения.

При наличии в папке с книгами подпапки c метаданными из первой версии KyBook по имени #metadata#, метаданные (название, авторы, серия и т.п.) будет взяты из метаданных первой версии а не получены из файла книги. В настоящий момент импорт ограничен, не все значения метаданных первой версии могут быть импортированы.
